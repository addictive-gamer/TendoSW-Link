Conectar cuenta Nintendo

Esta página genera el enlace de autorización correcto y extrae tu session_token de forma segura. Todo ocurre en tu navegador, ningún dato se envía a servidores externos.

1

Haz clic para ir al login de Nintendo

Se abrirá la página oficial de Nintendo en una nueva pestaña. Inicia sesión con tu cuenta bot.

2

Copia la URL de error

Después del login, el navegador mostrará un error porque la URL de regreso empieza con npf://. Copia esa URL completa de la barra de direcciones y pégala abajo.

3

Obtén tu session_token

Esta página extrae el código y hace la petición final a Nintendo para darte el token.

Pega la URL de regreso aquí

Es la URL que aparece en la barra de direcciones después del login. Empieza con npf71b963c1b7b6d119:// y el navegador muestra error — eso es normal.

✅ ¡session_token obtenido!

Guarda este token en un lugar seguro. Lo necesitarás para configurar TendoSW-Link. Haz clic para copiarlo:

👆 Clic para copiar
⚠️ No compartas este token con nadie. Da acceso completo a tu cuenta Nintendo. Úsalo solo en tu Worker de TendoSW-Link.
